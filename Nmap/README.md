# Nmap

This folder contains notes, screenshots, and practical exercises performed using Nmap in a personal cybersecurity lab.

## Topics Covered

- Host Discovery
- Port Scanning
- Service Enumeration
- OS Detection
- Version Detection
- Network Mapping

## Example Commands

```bash
nmap -sV <target>

nmap -A <target>

nmap -p- <target>
```

## Status

Learning and documenting practical Nmap usage.
