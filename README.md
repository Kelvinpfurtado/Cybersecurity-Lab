# Cybersecurity Lab

A personal cybersecurity learning lab built to gain practical experience with penetration testing, network analysis, vulnerability assessment, and security tools.

## Objectives

- Learn network scanning and enumeration
- Practice web application testing
- Analyze network traffic
- Understand common vulnerabilities
- Build hands-on cybersecurity skills

## Tools Covered

### Nmap
Network discovery, host enumeration, service detection, and port scanning.

### Wireshark
Packet capture, protocol analysis, and network troubleshooting.

### Burp Suite
Web application testing, request interception, and security assessment.

### Metasploit
Vulnerability validation and controlled exploitation within a lab environment.

## Lab Environment

- Kali Linux
- Windows Virtual Machines
- Metasploitable
- VMware Workstation
